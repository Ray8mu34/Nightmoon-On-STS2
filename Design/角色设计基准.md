# 卡池

当前各角色卡池，比起塔一的70张，大了不少：

| 角色          |  当前公开列表总数 |
| ----------- | -----: |
| Ironclad    | 90 张  |
| Silent      | 91 张  |
| Regent      | 88 张  |
| Necrobinder | 88 张  |
| Defect      | 92 张 |

按稀有度来分，只看白蓝金奖励池，一个标准角色大约是：20 白 + 36 蓝 + 26 金 = 82 张

| 稀有度              |       数量建议 | 设计含义                             |
| ---------------- | ---------: | -------------------------------- |
| Starter / Basic  |     3 张唯一卡 | 起始牌组用，通常包含 Strike、Defend、角色专属起始牌 |
| 白卡 Common        | **约 20 张** | 基础伤害、基础防御、角色机制入门                 |
| 蓝卡 Uncommon      | **约 36 张** | 机制扩展、条件牌、过渡型构筑核心                 |
| 金卡 Rare          | **约 26 张** | 构筑核心、强力 payoff、长期成长、稀有解法         |
| Ancient / 特殊高稀有度 |  **约 2 张** | STS2 新层级，高冲击、高稀有、通常不当作普通奖励牌设计    |

按卡牌类型来分，大概是这样：

| 类型        |     建议比例 | 
| --------- | ---------: | 
| 攻击 Attack | **35–45%** |   
| 技能 Skill  | **35–45%** | 
| 能力 Power  | **15–25%** |
