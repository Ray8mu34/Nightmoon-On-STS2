# 药剂师机制说明

## 定位

- 角色定位：`状态叠加`、`强效爆发`、`消耗回收`、`调合连锁`
- 核心体验：先铺设不同药效，再用 `强效` 放大药剂牌，最后靠 `鸡尾酒`、`终极鸡尾酒`、`药剂风暴` 之类的连段完成爆发
- 主要流派：`火毒调合`、`冰控弱化`、`雷暴连发`、`回收连锁`
- 设计目标：保留《月圆之夜》药剂师“药水、强效、消耗、回收、花式调配”的职业感觉，但改写成适配 `STS2` 单能量体系的多状态连锁角色

---

## 核心机制

### 1. 强效

- `强效` 是药剂师的核心自我增益状态
- 建议规则：
  - 当你打出1张带有 `药剂` 标签的牌时，消耗1层 `强效`
  - 该药剂牌获得以下增益：
    - 若造成直接伤害，额外造成 `3` 点伤害
    - 若施加负面状态，额外施加 `2` 层
    - 若回复生命或获得格挡，额外回复或获得 `2`
- 若1张药剂牌同时包含多种效果，则所有对应部分都能吃到 `强效` 加成
- `强效` 不替代能量，不应该变成第二条能量条；它更像“精炼后的药性储备”

### 2. 药剂牌

- `药剂` 是药剂师的核心标签，代表真正的配方成品
- 药剂牌一般分为几类：
  - `火焰药剂`：偏持续伤害和终结
  - `毒素药剂`：偏叠层和拖回合
  - `冰霜药剂`：偏控制和减伤
  - `雷电药剂`：偏多段与连击
  - `回复药剂`：偏生存和续航
  - `暗影药剂`：偏穿刺、吸血、爆发
- 大部分强牌都不是单张裸打，而是建立在“药剂牌数量”和“不同药效覆盖率”之上

### 3. 药效

- 药剂师最核心的“牌组奖励”不是单一状态，而是 `不同药效并存`
- 这版设计里主要使用四个核心药效：
  - `中毒`
  - `烧伤`
  - `寒冷`
  - `黏着`
- 建议规则：
  - `中毒`：沿用常见持续伤害逻辑，回合结束受到等同层数伤害并递减
  - `烧伤`：目标回合结束时受到等同层数伤害，然后失去一半层数，偏爆发型 DOT
  - `寒冷`：每层使目标本回合攻击伤害 `-1`，回合结束失去一半层数
  - `黏着`：每层使目标本回合攻击伤害 `-1`，并在其获得格挡时减少同值格挡，回合结束失去1层
- `黏着` 的定位不是再做一个中毒，而是让药剂师拥有区别于女巫冰霜的“化学性控制”

### 4. 调合

- `调合` 不是一个单独资源，而是一种构筑逻辑
- 牌组会不断问两个问题：
  - 目标身上有几种不同负面状态
  - 你这场战斗已经用过几类不同药剂
- `鸡尾酒`、`多重感染`、`药性分析`、`终极鸡尾酒` 都会围绕这个结构给奖励
- 这决定了药剂师不是“只堆中毒”就能满配发挥，而是更鼓励混色药效

### 5. 消耗与回收

- 月圆之夜药剂师有非常强的“移除牌当原料”的味道，这里统一翻译成 `消耗` 与 `回收`
- 药剂师会主动把许多牌打成 `消耗`
- 但同时又能通过 `回收装置`、`回忆`、`翻腾`、`原始法则` 等方式把消耗变成收益
- 设计意义：
  - 让药剂师的爆发不是凭空出现，而是来自提前准备过的原料链条
  - 让“临时牌”与“消耗牌”在药剂师手里不是损失，而是资产

## 角色玩法骨架

### 1. 前期节奏

- 用 `摇晃`、`实验改良`、`魔药强效`、`血清过载` 之类的牌建立第一批 `强效`
- 用 `燃烧药水`、`冰冷药水`、`毒雾试剂`、`粘性药水` 尽快把药效挂上去
- 这阶段药剂师的关键不是单卡伤害，而是尽早让敌人开始“带着多种状态过回合”

### 2. 中期转折

- 当你拿到 `快速混合`、`浓缩`、`鸡尾酒`、`女巫坩埚`、`风箱`、`蓄电池` 后，牌组会明显进入连段期
- 药剂师在这个阶段开始变成“先用配方牌做准备，再用药剂牌成批结算”
- 正确玩法不是见药就喝，而是判断：
  - 这回合是铺状态
  - 还是要把强效一次性转成爆发

### 3. 后期终结

- 后期常见终结方式有四种：
  - `剧毒蒸馏 + 鸡尾酒 / 终极鸡尾酒`
  - `强效膨胀 + 药剂风暴`
  - `对撞机 + 反物质 / 超级反物质`
  - `原始法则 + 翻腾 / 腐败 / 湮灭` 的消耗连锁

## 流派分支

### 1. 火毒调合线

- 关键词：`烧伤`、`中毒`、`多状态收益`
- 核心牌：
  - `燃烧药水`
  - `毒雾试剂`
  - `鸡尾酒`
  - `多重感染`
  - `剧毒蒸馏`
  - `终极鸡尾酒`
  - `风箱`
- 玩法特征：
  - 不追求单一状态极限，而是追求火毒并存后的复合收益
  - 对高血量精英和 Boss 很强

### 2. 冰控弱化线

- 关键词：`寒冷`、`黏着`、`拖节奏`
- 核心牌：
  - `冰冷药水`
  - `粘性药水`
  - `粘性炸弹`
  - `缩小射线`
  - `中和剂`
  - `加护`
- 玩法特征：
  - 通过多种减伤和弱化药效拖长战斗
  - 爆发不如火毒，但稳定性高

### 3. 雷暴连发线

- 关键词：`多段伤害`、`打点堆叠`、`药剂连发`
- 核心牌：
  - `雷暴药水`
  - `闪电链接`
  - `风筝实验`
  - `蓄电池`
  - `闪电爆破`
  - `雷阵雨`
  - `雷暴连锁`
- 玩法特征：
  - 类似“化学机枪”路线
  - 很吃出牌密度和雷电触发次数

### 4. 回收连锁线

- 关键词：`消耗收益`、`资源转化`、`长线滚雪球`
- 核心牌：
  - `研磨`
  - `回收装置`
  - `回忆`
  - `翻腾`
  - `腐败`
  - `湮灭`
  - `原始法则`
  - `女巫坩埚`
- 玩法特征：
  - 这条线最有月圆之夜药剂师原味
  - 需要更理解牌序和资源循环，但成型后上限极高

## 初始遗物

### 便携坩埚

- 效果：
  - 战斗开始时，获得 `1` 层 `强效`
  - 每回合第一次有牌被你主动 `消耗` 时，抽1张牌
- 设计理由：
  - 第一条保证药剂师开局就能打出“强化药剂”的职业手感
  - 第二条让角色从一开始就知道“消耗不是损失，而是资源发动机”
  - 效果偏稳定，而不是直接送钱或直接复制，适合作为初始遗物

### 遗物升级方向

- 若后面要扩专属遗物池，可以给 `便携坩埚` 两个升级方向：
  - `稳定型`：每回合第一次药剂牌额外获得1层临时强效
  - `连锁型`：每回合第一次消耗药剂牌时，将1张随机药剂牌置入弃牌堆

## 升级分支

这里的“升级分支”指角色在构筑和拿牌上的中期发展路线，不是单张牌的 `+` 升级表。

### 1. 火毒调合线

- 适合起手拿到：`燃烧药水`、`毒雾试剂`、`鸡尾酒`
- 升级优先：
  - 先补 `稳定挂状态`
  - 再补 `调合收益`
  - 最后补 `终极鸡尾酒 / 剧毒蒸馏` 这类终结牌
- 成型标志：
  - 对单个目标能稳定维持至少2到3种药效并持续转化为爆发

### 2. 冰控弱化线

- 适合起手拿到：`冰冷药水`、`粘性药水`、`加护`
- 升级优先：
  - 先补 `寒冷 / 黏着覆盖`
  - 再补 `防御与转化`
  - 最后补 `中和剂` 或多状态收益牌
- 成型标志：
  - 敌人的高伤意图能被寒冷和黏着明显压低

### 3. 雷暴连发线

- 适合起手拿到：`雷暴药水`、`闪电链接`、`蓄电池`
- 升级优先：
  - 先补 `低费雷电药剂`
  - 再补 `多段触发引擎`
  - 最后补 `闪电爆破 / 雷暴连锁`
- 成型标志：
  - 你能在1回合内稳定触发多次雷电结算，并把这些次数转成额外伤害

### 4. 回收连锁线

- 适合起手拿到：`研磨`、`回收装置`、`回忆`
- 升级优先：
  - 先补 `低成本消耗手段`
  - 再补 `消耗收益能力牌`
  - 最后补 `湮灭 / 原始法则 / 腐败`
- 成型标志：
  - 你已经不怕牌被消耗，反而希望每回合主动消耗牌来扩大收益

## 建议初始卡组

- `投瓶` x4
- `应急敷料` x4
- `燃烧药水` x1
- `摇晃` x1

说明：

- 这套初始卡组重点不是让药剂师一开场就拥有完整调合系统，而是先给他最基础的职业辨识度
- `投瓶` 保证通用输出
- `应急敷料` 保证基础防守
- `燃烧药水` 让玩家第一层就认识“药剂牌”和状态体系
- `摇晃` 让玩家立刻接触 `强效`

---

## 平衡注意点

- `强效` 一定要强，但不能强到替代能量体系，否则药剂师会变成“存了几层就随便横扫”
- `浓缩`、`提纯装置`、`药剂风暴`、`强效膨胀` 这类牌的叠乘很危险，需要重点测试
- `烧伤` 和 `中毒` 不能功能完全重合，否则药效种类看起来多，实际玩法会塌缩成一种 DOT
- `回收` 体系如果回复太稳定，容易让药剂师同时拥有高爆发、高续航、高资源循环，需要控制回忆类卡的密度
- `原始法则` 是非常像“先古级”核心的卡，数值和触发次数最好在实装时谨慎收敛
