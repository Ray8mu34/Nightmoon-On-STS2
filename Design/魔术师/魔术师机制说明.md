# 魔术师机制说明

## 定位

- 角色定位：`复制连锁`、`预演资源`、`伏笔埋设`、`反制控场`
- 核心体验：先用戏法牌整理手牌和资源，再通过 `预演` 压低咒术成本或放大咒术，最后靠 `复制`、`鬼牌`、`反制` 和 `伏笔` 打出连续爆发
- 主要流派：`咒术连发`、`伏笔爆破`、`反制控制`、`复制戏法`
- 设计目标：保留《月圆之夜》魔术师“舞台戏法、复制法术、恶作剧和拆招”的职业味道，但不依赖“敌方牌组”这一在 `STS2` 中不存在的系统

---

## 先说难点

### 1. 为什么魔术师难改

- 月圆之夜魔术师的很多牌，原本都和 `敌方牌组`、`恶作剧`、`偷牌`、`反制敌方出牌` 强绑定
- 但在 `STS2` 里：
  - 敌人通常没有一个能被你操作的公开牌组
  - 你也不会像原作那样去“塞牌进敌方牌库再等对面抽到”
  - 反制也不能直接照抄成“敌方打出某类牌时无效”

### 2. 这版的解决方案

- 我把“塞进敌方牌组的恶作剧”统一翻译成了 `伏笔`
- 我把“针对敌方出牌的反制”统一翻译成了 `基于敌方意图和攻击行为的延迟触发`
- 我把原作法力体系翻译成了 `预演`

所以这版魔术师的逻辑不是：

- “往敌人牌组里塞垃圾，等他抽到”

而是：

- “在敌人身上埋下伏笔”
- “提前布置反制”
- “在自己回合靠预演、复制和鬼牌完成舞台连段”

---

## 核心机制

### 1. 预演

- `预演` 是魔术师的战斗内资源
- 默认规则：
  - 战斗开始时 `预演 = 0`
  - 回合结束时 `预演不会清空`
  - `预演` 没有默认硬上限，但建议测试时重点关注 `6` 以上的爆发区间
- `预演` 的用途：
  - 作为阈值：例如“若你有预演”
  - 作为减费来源：配合 `魔术帘幕`、`清晰术`、`吹气`
  - 作为高费咒术前的准备资源：例如 `念力爆破`
- 设计意义：
  - 它承接了原作“法力越攒越能玩花活”的感觉
  - 但不直接替代能量，因此不会破坏 `STS2` 的基础节奏

### 2. 伏笔

- `伏笔` 是加在敌人身上的专属负面状态
- 建议规则：
  - 敌方回合开始时，若其有 `伏笔`，失去1层 `伏笔` 并受到2点伤害
- 设计定位：
  - 它是原作 `恶作剧 / 塞牌 / 陷阱` 的综合翻译
  - 不是纯粹的持续伤害，而是“你在敌人身上提前埋好的舞台机关”
- 与卡牌联动方式：
  - 有些牌负责铺 `伏笔`
  - 有些牌根据 `伏笔` 层数直接打高伤
  - 有些牌会消耗所有 `伏笔` 完成终结

### 3. 反制

- `反制` 是魔术师最重要的防御和花式操作体系之一
- 在这版里，`反制牌` 不再要求“敌方打出攻击牌/行动牌/咒术牌”
- 统一改写为：
  - “直到你下回合开始，若敌人攻击……”
  - 或“若敌人本回合意图攻击……”
  - 或“若敌人本回合不意图攻击……”
- 设计意义：
  - 这样既保留了“拆招”“埋伏”“预判”的味道
  - 又完全兼容 `STS2` 的意图系统

### 4. 复制与鬼牌

- `复制` 是魔术师的上限来源之一
- 复制主要来自：
  - `镜像`
  - `镜像手法`
  - `突发奇想`
  - `共鸣`
- `鬼牌` 则负责保留原作“上一张咒术的替身”感觉
- 建议规则：
  - `鬼牌` 视为你本回合打出的上一张 `咒术牌`
  - 若没有可复制目标，则提供最低限度的过牌补偿

## 角色玩法骨架

### 1. 前期节奏

- 用 `筹谋`、`预演`、`舞台灯光`、`魔术帘幕` 建立第一批 `预演`
- 用 `伏线`、`小丑戏法`、`恶魔手镜` 先把 `伏笔` 埋到敌人身上
- 同时用 `烟幕步`、`退缩`、`凝视` 这类牌稳住前期防御

### 2. 中期转折

- 当你拿到 `鬼牌`、`镜像`、`传送门`、`帽子戏法`、`魔术力场` 之后，魔术师会进入真正的表演期
- 这阶段的重点判断是：
  - 这一回合是先铺 `伏笔`
  - 还是先留 `反制`
  - 还是把 `预演` 一口气转成咒术爆发
- 正确玩法不是所有系统都平均推进，而是围绕你当前抽到的核心件做一轮偏重构筑

### 3. 后期终结

- 后期常见终结方式有四种：
  - `预演 + 咒术` 的大回合，例如 `念力爆破`
  - `悬浮 / 共鸣 / 鬼牌` 的复制连段
  - `伏笔铺场 + 恶作剧惊魂 / 背叛 / 偷天换日`
  - `反制引擎 + 连锁机关 / 魔术力场 / 备用牌盒`

## 流派分支

### 1. 咒术连发线

- 关键词：`预演`、`减费`、`重复结算`
- 核心牌：
  - `预演`
  - `魔术帘幕`
  - `传送门`
  - `悬浮`
  - `共鸣`
  - `念力爆破`
  - `终幕魔术`
- 玩法特征：
  - 这是最接近原作“攒蓝施法”的路线
  - 先准备，再把一串咒术在同回合打干净

### 2. 伏笔爆破线

- 关键词：`埋机关`、`结算层数`、`终结一击`
- 核心牌：
  - `伏线`
  - `小丑戏法`
  - `恶魔手镜`
  - `娴熟魔术`
  - `恶作剧惊魂`
  - `背叛`
  - `偷天换日`
- 玩法特征：
  - 是对“恶作剧牌组”的最核心翻译
  - 玩起来更像魔术师提前布好机关，再一口气引爆全场

### 3. 反制控制线

- 关键词：`预判敌意图`、`拆招`、`反打`
- 核心牌：
  - `烟幕步`
  - `帽子戏法`
  - `布满陷阱`
  - `木偶戏法`
  - `连锁机关`
  - `退缩`
  - `反冲`
- 玩法特征：
  - 这条线最强调“舞台表演中的预判和拆招”
  - 对高攻敌人和大动作 Boss 有很强针对性

### 4. 复制戏法线

- 关键词：`复制`、`鬼牌`、`价值翻倍`
- 核心牌：
  - `鬼牌`
  - `镜像手法`
  - `镜像`
  - `突发奇想`
  - `幕后主使`
  - `共鸣`
  - `复制道具`
- 玩法特征：
  - 这条线最花哨，也最有“魔术师就是玩骗术和手法”的味道
  - 牌序要求高，但乐趣非常足

## 初始遗物

### 空帽戏箱

- 效果：
  - 战斗开始时，获得 `1` 点 `预演`
  - 每回合第一次施加 `伏笔` 时，抽1张牌
- 设计理由：
  - 第一条让魔术师从开局就有“我在搭舞台”的手感
  - 第二条鼓励玩家主动埋 `伏笔`，而不是只当普通法师玩
  - 整体偏稳定启动，不会直接给出过量爆发

### 遗物升级方向

- 若后续扩专属遗物池，可以给 `空帽戏箱` 两个升级方向：
  - `铺场型`：每回合第一次施加伏笔时，额外施加1层
  - `施法型`：每回合第一次获得预演时，再让1张咒术牌费用-1

## 升级分支

这里的“升级分支”指角色在构筑和拿牌上的中期发展路线，不是单张牌的 `+` 升级表。

### 1. 咒术连发线

- 适合起手拿到：`预演`、`雷鸣`、`魔术帘幕`
- 升级优先：
  - 先补 `稳定预演`
  - 再补 `减费和过牌`
  - 最后补 `悬浮 / 念力爆破 / 终幕魔术`
- 成型标志：
  - 你能稳定把两张以上中高费咒术塞进同一个大回合

### 2. 伏笔爆破线

- 适合起手拿到：`伏线`、`小丑戏法`、`恶魔手镜`
- 升级优先：
  - 先补 `稳定铺伏笔`
  - 再补 `伏笔收益牌`
  - 最后补 `恶作剧惊魂 / 背叛 / 偷天换日`
- 成型标志：
  - 伏笔不再只是顺手挂一下，而是已经成为明确的主要伤害来源

### 3. 反制控制线

- 适合起手拿到：`烟幕步`、`退缩`、`布满陷阱`
- 升级优先：
  - 先补 `基础反制密度`
  - 再补 `反制触发收益`
  - 最后补 `连锁机关 / 备用牌盒 / 反冲`
- 成型标志：
  - 敌人的关键攻击回合已经会变成你的优势回合

### 4. 复制戏法线

- 适合起手拿到：`鬼牌`、`镜像手法`、`水晶球`
- 升级优先：
  - 先补 `低费高质量目标`
  - 再补 `复制引擎`
  - 最后补 `共鸣 / 幕后主使 / 突发奇想`
- 成型标志：
  - 你的关键牌不再只打一遍，而是能靠复制、鬼牌和重复结算滚出巨大价值

## 建议初始卡组

- `纸牌飞掷` x4
- `帷幕闪避` x4
- `筹谋` x1
- `伏线` x1

说明：

- `纸牌飞掷` 提供基础输出
- `帷幕闪避` 让基础防御就和敌方意图互动
- `筹谋` 提供最早的 `预演`
- `伏线` 则明确告诉玩家：这个角色不是单纯法师，而是会在敌人身上埋戏法的人

---

## 平衡注意点

- `预演` 很容易和减费叠出超高收益，所以高费咒术的数值必须保守测试
- `反制` 如果做得太强，会让魔术师对攻击型敌人形成过度压制
- `伏笔` 既要能承接原作恶作剧味道，又不能只是换皮中毒，因此它最好更多和“爆破牌”绑定
- `复制` 和 `额外生效` 的强度叠乘非常危险，特别要关注 `悬浮 + 共鸣 + 鬼牌 + 终幕魔术`
- `偷天换日` 是很典型的先古级清场卡，数值应按高难环境反复收敛
