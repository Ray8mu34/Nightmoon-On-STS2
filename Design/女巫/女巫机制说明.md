# 女巫机制说明

## 定位

- 角色定位：`法术连锁`、`资源积累`、`属性伤害`、`回合爆发`
- 核心体验：前几回合通过 `蓄法`、检索、减费和额外生效做准备，中后段打出高质量咒术连段
- 主要流派：`火焰增伤`、`冰霜寒冷`、`毒转寒`、`蓄法风暴`
- 设计目标：保留《月圆之夜》女巫“法力越滚越大、法术越打越离谱”的感觉，但完全适配 `STS2` 的单能量体系

---

## 核心机制

### 1. 蓄法

- `蓄法` 是女巫的专属战斗内资源，不占用能量，也不直接替代费用
- 默认规则：
  - 战斗开始时 `蓄法 = 0`
  - 回合结束时 `蓄法不会清空`
  - `蓄法` 没有默认上限，若后续测试发现爆发过高，可加上软上限 `10`
- `蓄法` 的作用主要有三种：
  - 作为阈值：例如“若你有蓄法”或“若你有 3 点以上蓄法”
  - 作为消耗资源：例如 `飓风`、`法力风暴`
  - 作为转化媒介：把月圆之夜里的“法力积累”翻译为 `减费`、`额外生效`、`防御转换`

### 2. 寒冷

- `寒冷` 是女巫最重要的负面状态之一
- 建议规则：
  - 敌人每层 `寒冷` 使其本回合攻击伤害 `-1`
  - 敌人回合结束时，失去一半 `寒冷`，向下取整
- 设计意义：
  - 它既是输出前置，也是一种偏控制型的防御手段
  - `冰霜` 体系不是单纯叠伤，而是通过削弱敌方攻击频率和幅度换节奏
- 与女巫牌的联动：
  - 有些牌只要求目标“已有寒冷”
  - 有些牌会“结算寒冷一次”或根据寒冷翻倍伤害

### 3. 咒术

- `咒术` 是女巫的主战牌标签，不等于卡牌类型，而是构筑标签
- 女巫大量牌会关心：
  - 本回合是否已打出咒术
  - 本场战斗打出过多少张咒术
  - 下一张咒术牌是否减费
  - 下一张咒术牌是否额外生效
- 这让女巫更像一个“法术连段角色”，而不是传统的单张大招角色

### 4. 额外生效

- 月圆之夜女巫有很多“法术强化”“重复触发”味道，这里统一翻译为 `额外生效`
- 建议规则：
  - 若一张牌“额外生效一次”，则按原目标再完整结算一次
  - 若牌本身带随机目标，则第二次重新随机
  - 若牌带有 `消耗`，结算完成后仍只消耗一次
- 这个关键词是女巫的上限来源，也是强度最需要反复测的部分

## 角色玩法骨架

### 1. 前期节奏

- 用 `智慧`、`灵感`、`法力冰霜`、`法力烈焰`、`痛苦打击` 等牌稳定拿到第一批 `蓄法`
- 同时用 `预言`、`水晶球`、`法术预言` 整理手牌质量
- 前期的女巫不追求瞬间斩杀，而是追求“下回合一定能打漂亮”

### 2. 中期转折

- 当牌组出现 `快速施法`、`咒语强化`、`法力膨胀`、`回音`、`记忆增幅` 这类牌后，女巫开始进入连段期
- 这时要学会区分：
  - 哪些回合是 `铺资源回合`
  - 哪些回合是 `爆发回合`
- 女巫很怕“每回合都想做所有事”，正确玩法是有意识地储备 `蓄法` 和关键咒术

### 3. 后期终结

- 后期终结方式通常有四种：
  - `炎爆术 / 火球 / 火山喷发` 的火焰爆杀
  - `冰霜新星 / 致死霜冻 / 灾难` 的寒冷压制
  - `枯萎 + 毒爆 + 毒性蔓延` 的持续结算
  - `法力之潮 / 能量饮料 / 飓风 / 法力风暴` 的蓄法风暴回合

## 流派分支

### 1. 火焰分支

- 关键词：`持续增伤`、`中后期爆发`、`多段火伤`
- 核心牌：
  - `火苗`
  - `法力烈焰`
  - `火球`
  - `火山喷发`
  - `炎爆术`
  - `烧焦`
  - `火把`
  - `燃烧果实`
- 玩法特征：
  - 先把每张火焰牌的质量垫高，再用高倍率大法术收头
  - 对单爆发最稳定，适合打精英和 Boss

### 2. 冰霜分支

- 关键词：`寒冷控制`、`节奏压制`、`条件爆发`
- 核心牌：
  - `寒流`
  - `冰盾`
  - `凝固`
  - `暴风雪`
  - `致死霜冻`
  - `冰霜新星`
  - `冰霜护符`
  - `灾难`
- 玩法特征：
  - 先让敌人的攻击回合变钝，再用“若目标已有寒冷”的高收益牌完成压制
  - 对高伤敌人和多段攻击敌人特别有效

### 3. 毒转寒分支

- 关键词：`双状态联动`、`持续伤害`、`拖回合获胜`
- 核心牌：
  - `酸性喷雾`
  - `枯萎`
  - `毒爆`
  - `毒性蔓延`
  - `寒流`
  - `暴风雪`
- 玩法特征：
  - 不是单纯堆毒，而是利用“中毒转寒冷”把进攻和控制绑在一起
  - 对高血量敌人的成长性最好

### 4. 蓄法风暴分支

- 关键词：`资源囤积`、`大回合`、`法术连锁`
- 核心牌：
  - `智慧`
  - `灵感`
  - `法力膨胀`
  - `法力之潮`
  - `能量饮料`
  - `飓风`
  - `法力风暴`
  - `法力源泉`
- 玩法特征：
  - 这是最像月圆之夜女巫的路线
  - 需要更高的牌序、资源判断和回合规划，但上限也最高

## 初始遗物

### 星界法典

- 效果：
  - 战斗开始时，获得 `1` 点 `蓄法`
  - 你每回合打出的第一张 `咒术牌` 额外获得 `1` 点 `蓄法`
- 设计理由：
  - 让女巫从第一回合就有“法力职业”的手感
  - 避免角色在未抽到核心蓄法牌时完全空转
  - 强度集中在“稳定启动”，而不是直接给爆发，适合作为初始遗物

### 遗物升级方向

- 若后面要做角色专属遗物池，可以让 `星界法典` 的上位版本往两个方向走：
  - `稳定型`：每回合第一张咒术额外抽1
  - `爆发型`：当你首次达到 5 点蓄法时，本回合下一张咒术额外生效一次

## 升级分支

这里的“升级分支”指角色在构筑和拿牌上的中期发展路线，不是单张牌的 `+` 升级数值表。

### 1. 火焰爆发线

- 适合起手拿到：`火苗`、`火球`、`法力烈焰`、`烧焦`
- 升级优先：
  - 先补 `稳定蓄法`
  - 再补 `火焰增伤`
  - 最后补 `大法术终结`
- 成型标志：
  - 单回合能稳定打出 `1张强化牌 + 1张中型火法 + 1张终结牌`

### 2. 冰霜控制线

- 适合起手拿到：`寒流`、`冰盾`、`凝固`、`暴风雪`
- 升级优先：
  - 先把 `寒冷覆盖率` 做出来
  - 再补 `寒冷收益牌`
  - 最后补 `AOE终结`
- 成型标志：
  - 大多数敌人的强攻击回合都能被寒冷显著削弱

### 3. 毒转寒运营线

- 适合起手拿到：`酸性喷雾`、`毒爆`、`毒性蔓延`
- 升级优先：
  - 先保证 `中毒来源`
  - 再补 `转寒冷` 和 `即时结算`
  - 最后补高质量防御和检索
- 成型标志：
  - 牌组已经不需要靠普通直伤赢，而是靠状态滚雪球压死敌人

### 4. 蓄法风暴线

- 适合起手拿到：`智慧`、`灵感`、`法力膨胀`、`法力之潮`
- 升级优先：
  - 先拿 `稳定蓄法`
  - 再拿 `减费 / 额外生效`
  - 最后拿 `飓风 / 法力风暴 / 炎爆术` 这类终结牌
- 成型标志：
  - 你已经能明确区分“准备回合”和“风暴回合”
  - 关键回合中，能让多张咒术在同回合连续结算

## 建议初始卡组

- `火花术` x4
- `凝霜术` x4
- `智慧` x1
- `寒流` x1

说明：

- 初始卡组故意不直接塞入高爆发牌，避免女巫开局就变成“抽到关键牌才玩游戏”
- `火花术` 保证最基本的对单输出
- `凝霜术` 让基础防御也带一点职业辨识度
- `智慧` 提供最早的 `蓄法` 手感
- `寒流` 让玩家在第一层就知道这个角色和 `寒冷` 强绑定

---

## 平衡注意点

- `蓄法` 强但不该等于第二条能量条，否则会直接破坏 `STS2` 的费用节奏
- `额外生效` 的强度很容易失控，因此最好优先限制在：
  - 只作用于咒术
  - 一回合内有限次数
  - 高价值触发需要先蓄法或先铺垫
- `寒冷` 如果同时兼顾高减伤和高爆发，容易变成全能机制，所以建议它更偏 `节奏控制`
- `毒转寒` 很有特色，但要小心不要让它完全挤掉纯冰或纯火构筑
