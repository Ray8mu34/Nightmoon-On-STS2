# 契约师机制说明

## 定位

- 角色定位：`高风险爆发`、`战斗内资源积累`、`失血转收益`、`回收连锁`
- 核心体验：先用 `审判` 和 `恐惧` 控场，再通过 `献祭` 失血把资源和爆发推高，最后用高倍率契约牌完成收割
- 主要流派：`审判积累`、`献祭爆发`、`恐惧控制`、`灵魂回收`
- 设计目标：保留《月圆之夜》契约师“以命换力、越危险越强、靠死亡和恐惧压制敌人”的感觉，同时适配 `STS2` 的单能量体系

---

## 核心机制

### 1. 审判

- `审判` 是契约师的核心战斗内资源
- 默认规则：
  - 战斗开始时 `审判 = 0`
  - 回合结束时 `审判不会清空`
  - `审判` 没有硬上限，但建议测试时关注 `10` 以上的爆发区间
- `审判` 的用途：
  - 作为阈值：例如“若你有5点以上审判”
  - 作为结算倍率：例如 `魂飞魄散`
  - 作为消耗资源：部分终结技和引擎牌会失去 `审判` 来触发额外效果
- 设计意义：
  - 它替代了原作里“越玩越深、越来越危险”的契约积累感
  - 不是第二条能量条，而是逐回合累积的灾厄资本

### 2. 献祭

- `献祭` 不是单独资源，而是一种判定
- 建议规则：
  - 你在自己的回合因为牌的效果失去生命时，视为 `献祭`
  - 每次独立失去生命都算1次 `献祭`
- 常见收益：
  - 获得 `审判`
  - 获得格挡或回复
  - 提升本回合伤害
  - 触发能力牌，如 `冥界花`、`灵魂锁链`
- 设计意义：
  - 契约师必须把血量当作资源，而不是单纯的容错
  - 这样角色才能真正有“玩命换上限”的味道

### 3. 恐惧

- `恐惧` 是契约师的关键负面状态
- 建议规则：
  - 敌人在自己的回合攻击时，攻击伤害减少等同于 `恐惧` 层数，最多减少 `8`
  - 敌人回合结束时，失去一半 `恐惧`，向下取整
- 设计定位：
  - 它不是中毒那样的持续输出，而是偏 `压制攻击意图`
  - 同时大量契约牌还会要求“目标有恐惧”作为增伤或追加效果条件
- 玩法意义：
  - 契约师不是只有卖血，还要先把敌人吓住，才能安全地卖血

### 4. 灵魂与回收

- 月圆之夜契约师有很多“移除、还魂、招魂、献祭灵魂”的味道
- 这里统一翻译为 `消耗堆交互`
- 契约师的很多关键牌会：
  - 主动 `消耗`
  - 从 `消耗堆` 回收牌
  - 根据被消耗的牌数量或质量获得收益
- 设计意义：
  - 让契约师既有爆发，也有“死过的牌会回来”的诡异续航感
  - 这条线和药剂师的回收不同，契约师更偏“亡者复用”和高风险咒术循环

## 角色玩法骨架

### 1. 前期节奏

- 用 `审判低语`、`炼狱之审判`、`阴云密布`、`引魂刺` 建立第一批 `审判`
- 用 `恐惧低语`、`恐惧障壁`、`溃散` 让敌人的攻击回合先变钝
- 前期的契约师并不着急搏命，而是先把 `审判` 和 `恐惧` 两套底子铺起来

### 2. 中期转折

- 当你拿到 `冥界花`、`审判回路`、`灵魂摆渡人`、`能量吸收`、`血契研究` 之后，契约师开始进入真正的资源转换期
- 这阶段的判断重点是：
  - 这回合能不能安全 `献祭`
  - 献祭之后能不能马上把收益兑现
  - 是否值得为了下回合更强的 `审判` 再忍一轮

### 3. 后期终结

- 后期常见终结方式有四种：
  - `审判积累 + 魂飞魄散 / 冥王裁决`
  - `献祭爆发 + 危险契约 / 复仇怨魂 / 血债回响`
  - `恐惧控制 + 暗夜审判 / 死线预兆`
  - `灵魂回收 + 共鸣 / 契约轮回 / 冥河摆渡`

## 流派分支

### 1. 审判积累线

- 关键词：`资源滚雪球`、`阈值收益`、`大招终结`
- 核心牌：
  - `审判低语`
  - `炼狱之审判`
  - `秘密研究`
  - `审判回路`
  - `血色之审判`
  - `魂飞魄散`
  - `虚无之口`
- 玩法特征：
  - 前中期节奏偏慢，但只要审判堆起来，后期单牌质量极高
  - 是最稳定、也最容易做出职业辨识度的主线

### 2. 献祭爆发线

- 关键词：`卖血换上限`、`一回合爆杀`、`高风险`
- 核心牌：
  - `血税`
  - `血契研究`
  - `献祭仪式`
  - `猩红绽放`
  - `危险契约`
  - `复仇怨魂`
  - `血债回响`
- 玩法特征：
  - 契约师最刺激的一条线
  - 操作空间大，但容错最低

### 3. 恐惧控制线

- 关键词：`压低敌方输出`、`安全卖血`、`条件斩杀`
- 核心牌：
  - `恐惧低语`
  - `恐惧障壁`
  - `恐惧来袭`
  - `幽闭恐惧症`
  - `恐惧深渊`
  - `绝望蔓延`
  - `死线预兆`
- 玩法特征：
  - 先把敌人的攻击和节奏打乱，再从容积攒审判或组织献祭回合
  - 对高攻 Boss 和多段攻击敌人尤其重要

### 4. 灵魂回收线

- 关键词：`消耗交互`、`循环复用`、`诡异续战`
- 核心牌：
  - `招魂术`
  - `灵魂碎片`
  - `回忆`
  - `还魂术`
  - `灵魂摆渡人`
  - `契约轮回`
  - `冥河摆渡`
  - `黄泉之门`
- 玩法特征：
  - 这条线最接近原作“死去的牌也还会回来”的味道
  - 真正成型后，契约师的关键牌很难被一次性耗尽

## 初始遗物

### 断罪残页

- 效果：
  - 战斗开始时，获得 `1` 点 `审判`
  - 每回合第一次在自己回合 `献祭` 时，抽1张牌
- 设计理由：
  - 第一条让契约师从第一回合就有“积攒契约”的感觉
  - 第二条明确告诉玩家：失血不是纯负担，而是职业引擎
  - 该遗物提供的是 `稳定启动`，而不是直接给过量爆发，适合作为初始遗物

### 遗物升级方向

- 若后续扩专属遗物池，可以给 `断罪残页` 两个升级方向：
  - `稳态型`：每回合第一次获得审判时，额外获得2点格挡
  - `搏命型`：每回合第一次献祭时，额外获得1点审判

## 升级分支

这里的“升级分支”指角色在构筑和拿牌上的中期发展路线，不是单张牌的 `+` 升级表。

### 1. 审判积累线

- 适合起手拿到：`审判低语`、`炼狱之审判`、`断命书`
- 升级优先：
  - 先补 `稳定审判来源`
  - 再补 `审判阈值收益`
  - 最后补 `魂飞魄散 / 冥王裁决 / 虚无之口`
- 成型标志：
  - 你已经能稳定在中后段回合维持 `5` 到 `8` 点以上审判

### 2. 献祭爆发线

- 适合起手拿到：`血税`、`血契研究`、`献祭仪式`
- 升级优先：
  - 先补 `低成本失血`
  - 再补 `失血即时收益`
  - 最后补 `危险契约 / 复仇怨魂 / 血债回响`
- 成型标志：
  - 你能把一整回合的失血全部转成伤害、格挡或回复，而不是白白掉血

### 3. 恐惧控制线

- 适合起手拿到：`恐惧低语`、`恐惧障壁`、`溃散`
- 升级优先：
  - 先补 `恐惧覆盖率`
  - 再补 `恐惧收益牌`
  - 最后补 `死线预兆 / 绝望蔓延 / 暗夜审判`
- 成型标志：
  - 敌人的强攻击回合已经能被稳定压到你可接受的范围

### 4. 灵魂回收线

- 适合起手拿到：`招魂术`、`灵魂碎片`、`回忆`
- 升级优先：
  - 先补 `稳定消耗手段`
  - 再补 `回收引擎`
  - 最后补 `契约轮回 / 黄泉之门 / 冥河摆渡`
- 成型标志：
  - 你已经开始主动规划哪些牌要被消耗，而不是害怕消耗

---

## 建议初始卡组

- `血契斩` x4
- `恐惧祷言` x4
- `审判低语` x1
- `引魂刺` x1

说明：

- `血契斩` 让契约师从起手就接触“失血换资源”
- `恐惧祷言` 让基础防御也带职业辨识度
- `审判低语` 提供最早的 `审判` 资源
- `引魂刺` 让玩家在第一层就感受到“恐惧挂上去以后，别的牌会更强”

---

## 平衡注意点

- `审判` 很容易滚成超大资源池，所以后期大招一定要测试“倍率是不是过线”
- `献祭` 既要危险，也不能危险到没人敢用，因此需要给足即时回报
- `恐惧` 不能做成单纯的“另一种虚弱”，最好保持它和契约师专属牌之间的专有联动
- `灵魂回收` 体系如果回得太稳，契约师会同时拥有高爆发、高续航和无限循环，需要谨慎控制 `回忆 / 还魂术 / 黄泉之门`
- `最后的希望` 和 `终焉契约` 都是非常典型的高风险顶端卡，数值应以实战体验为准反复回调
